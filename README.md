## 📖 About

**SPACE WARS** is a neon‑drenched, endless space shooter built with vanilla JavaScript and powered by **Supabase** for authentication, cloud saves, real‑time multiplayer parties, and global leaderboards.

Fight through waves of enemies, survive intense boss battles, upgrade your ship, collect power‑ups, and compete with friends — all in a stunning cosmic battlefield.

## ✨ Features

- 🎮 **Endless Campaign** — Fight through infinite waves with escalating difficulty
- 👾 **Epic Boss Battles** — Multi‑phase bosses every 5 waves
- 🚀 **7 Unique Ships** — From the agile Viper to the legendary OMEGA and IAN'S SHIP
- 🛒 **Shop & Upgrades** — Buy ships, laser beams, shields, bombs, nukes, and extra lives
- 💾 **Cloud Saves** — Progress syncs across devices via Supabase
- 🔐 **Email & Google Sign‑In** — Secure authentication with automatic account linking
- 👥 **Friends System** — Add friends by username, invite them to parties
- 🎉 **Multiplayer Parties** — Create parties, ready up, and play together in real‑time
- 🏆 **Global Leaderboard** — Top 50 players ranked by hi‑score
- 🎨 **Dynamic Visuals** — Nebula backgrounds, shooting stars, planets, and particle effects
- 🔊 **Sound & Music** — Original soundtrack with togglable SFX
- 📱 **Mobile Support** — Auto‑detects touch controls on phones/tablets
- ⚙️ **Settings** — Resolution options (Auto, 720p, 1080p, 1440p) and control schemes

## 🕹️ Controls

| Key / Input | Action |
|-------------|--------|
| `Mouse` / `Touch` | Move ship |
| `WASD` / `Arrow Keys` | Move ship (alternative) |
| `Z` (Hold) | Laser beam (requires upgrade) |
| `B` | Bomb — clears screen |
| `N` | Nuke — massive boss damage |
| `ESC` | Pause / Return to menu |

## 🛠️ Tech Stack

- **Frontend:** Vanilla HTML5, CSS3, JavaScript (Canvas API)
- **Backend:** [Supabase](https://supabase.com) (Auth, Database, Realtime)
- **Fonts:** Orbitron, Rajdhani (Google Fonts)
- **Hosting:** GitHub Pages & Cloudflare Pages
